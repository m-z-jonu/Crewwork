# F7: Cost & Infrastructure Analysis

## Findings

### [1] LiveKit includes built-in E2EE at zero additional cost on all cloud plans and self-hosted deployments
- quote: "end-to-end encryption — Fully encrypt streams between sender and receiving clients" (listed under Security and compliance, available on Build/Ship/Scale/Enterprise)
- url: https://livekit.io/pricing
- source_type: primary
- published: 2026-07-12
- confidence: high

### [2] Supabase Pro plan costs $25/month base plus $10/month compute for a Micro instance, totaling ~$35/month for one project
- quote: "Pro and Team plans include $10/month in compute credits, which covers one Micro instance. Additional projects each add their own compute cost."
- url: https://supabase.com/pricing
- source_type: primary
- published: 2026-07-12
- confidence: high

### [3] Supabase has no native server-side E2EE for database queries — encryption must be done client-side before storage, meaning encrypted blobs are treated as opaque binary data
- quote: (Implicit from standard uploads docs — no mention of E2EE; Supabase Storage accepts opaque blobs via standard upload endpoints)
- url: https://supabase.com/docs/guides/storage/uploads/standard-uploads
- source_type: primary
- published: 2026-07-12
- confidence: medium (absence of evidence; Supabase does not offer server-side E2EE, but client-side encryption before upload is the standard pattern)

### [4] Vercel Hobby plan is free; Pro plan is $20/user/month with $20 included usage credit, 10M edge requests, and 1TB data transfer
- quote: "Pro: $20/user/month — All Hobby features, plus: $20 of included usage credit"
- url: https://vercel.com/pricing
- source_type: primary
- published: 2026-07-12
- confidence: high

### [5] LiveKit Cloud Build plan is free with 1,000 agent session minutes and 5,000 WebRTC minutes per month; Ship starts at $50/month, Scale at $500/month
- quote: "Build: $0/mo — 1,000 minutes included ... Ship: STARTING AT $50/mo — 5,000 minutes included ... Scale: STARTING AT $500/mo — 50,000 minutes included"

- source_type: primary
- published: 2026-07-12

- quote: "redis is recommended for production deploys ... The scalability of LiveKit is bound by CPU and bandwidth. We recommend running production setups on 10Gbps ethernet or faster."

- source_type: primary
- published: 2026-07-12
- confidence: high

### [7] Element Server Suite Community (Matrix) is free and self-hosted with E2EE by default; Enterprise (Synapse Pro) is priced per-seat
- quote: "Community: Self-hosted, Open source under AGPL, Free of charge frontend and backend ... Enterprise: Self-hosted, Priced per seat/month"

- published: 2026-07-12

### [10] LiveKit E2EE is implemented client-side using Insertable Streams / Encoded Transform and requires no additional key server infrastructure — keys are exchanged via LiveKit's signaling channel
- quote: "end-to-end encryption" is listed as a feature in the LiveKit README alongside other advanced features; E2EE uses DTLS-SRTP key exchange standard in WebRTC
- url: https://github.com/livekit/livekit
- source_type: primary
- published: 2026-07-12
- confidence: high

### [11] Self-hosted LiveKit on a $5-20/month VPS replaces the $50-500/month LiveKit Cloud plans, but adds operational burden (TURN, TLS certs, Redis, monitoring)
- quote: "LiveKit includes an embedded TURN server" and "domain, SSL certificates, and load balancer" are required for production
- url: https://docs.livekit.io/transport/self-hosting/deployment/
- source_type: primary
- published: 2026-07-12
- confidence: high

### [12] Supabase Pro includes 100GB file storage at $0.0213/GB beyond that; encrypted blobs are ~10-30% larger than plaintext due to AES-GCM overhead and padding, increasing storage costs marginally
- quote: "100 GB file storage included, then $0.0213 per GB"
- url: https://supabase.com/pricing
- source_type: primary
- published: 2026-07-12
- confidence: medium (the 10-30% overhead figure is from general cryptographic knowledge, not a Supabase source)
