# F2: WHMCS and Blesta hosting billing platforms

## Findings

### [1] WHMCS is the market leader with 35,000+ customers in 200+ countries, now on version 9.0 (GA January 20, 2026), owned by WebPros (IONOS/cPanel parent).
- quote: "Trusted by over 35,000 customers in over 200 countries"
- url: https://www.whmcs.com/
- source_type: primary
- published: 2026-01-20 (v9.0 announcement)
- confidence: high

### [2] WHMCS pricing is tiered by active client count, ranging from $34.95/mo (250 clients) to $1,999.95/mo (unlimited), with a cloud-hosted option starting at $9.99/mo for 25 clients.
- quote: "Plus $34.95 Per Month, Excl. VAT. Up to 250 Active Clients" / "Business Starting from $84.95 Per Month... Up to 500 Active Clients"
- url: https://www.whmcs.com/pricing/
- source_type: primary
- published: 2026
- confidence: high

### [3] Blesta is open-source (99%+ of code is open, only 3 files IonCube-encrypted for license), developed by Phillips Data Inc., current version 5.13.10, originally created in 2003 and publicly launched in 2007.
- quote: "Blesta is almost entirely open. Create extensions, or edit the core." / "Over 99% of Blesta's code is open... Only 3 files are encrypted for licensing purposes."
- url: https://www.blesta.com/ / https://www.blesta.com/compare/blesta-vs-whmcs/
- source_type: primary
- published: 2025
- confidence: high

### [4] Blesta pricing offers unlimited clients on all plans with flat rates: $17.95/mo (branded monthly), $20.95/mo (unbranded), or one-time purchase options from $350 to $500 (owned) and $1,295-$1,495 (lifetime).
- quote: "Monthly Branded $17.95 / per month - Unlimited clients & staff" / "Owned Branded $350 / one-time"
- url: https://www.blesta.com/pricing/
- source_type: primary
- published: 2025
- confidence: high

### [5] WHMCS is IonCube-encoded (closed-source core) which prevents independent security auditing; Blesta's near-open-source model allows full code inspection and modification.
- quote: "WHMCS: IonCube Encrypted - Cannot audit security implementations" vs "Blesta: Full access to view and modify core code"
- url: https://www.blesta.com/compare/blesta-vs-whmcs/
- source_type: primary (vendor comparison, but factually verifiable)
- published: 2025
- confidence: high

### [6] WHMCS provides a comprehensive RESTful API with OAuth support, hook system, and a beta API, while Blesta offers a RESTful API supporting JSON/XML/PHP with both inbound and outbound webhooks.
- quote (WHMCS): "The WHMCS API allows you to perform operations and actions within WHMCS from external third party and custom code." / Includes OAuth, Hooks, Internal API, and a Beta API.
- url: https://developers.whmcs.com/api/
- source_type: primary
- published: 2026
- confidence: high
- quote (Blesta): "RESTful API with JSON, XML, and PHP support" / "Bidirectional Webhooks: Both inbound and outbound webhook support"
- url: https://www.blesta.com/compare/blesta-vs-whmcs/
- source_type: primary
- confidence: high

### [7] WHMCS integrates with 15+ control panels, 40+ payment gateways, and 15+ domain registrars with a large marketplace of third-party modules; Blesta includes 12+ control panels, 39+ payment gateways, 11+ domain registrars built-in.
- quote (WHMCS): "Integrated with all the leading web hosting control panels and domain registrars"
- url: https://www.whmcs.com/
- source_type: primary
- confidence: high
- quote (Blesta): Lists cPanel, Plesk, DirectAdmin, VirtFusion, Enhance, CyberPanel, Virtualmin, Proxmox, Vultr, DigitalOcean, SolusVM, Pterodactyl as control panel integrations; Stripe, Authorize.net, PayPal, Square, Braintree, CoinGate, BTCPay as payment gateways.
- url: https://www.blesta.com/integrations/
- source_type: primary
- confidence: high

### [8] WHMCS offers a cloud-hosted option (WHMCS Cloud via WebPros) with automatic updates/backups and locations in Houston, Dallas, Düsseldorf, Zurich, and Tokyo; Blesta is self-hosted only.
- quote: "WHMCS Cloud delivers the full power of WHMCS, hosted, maintained, and always up to date" / Locations: Houston USA, Düsseldorf Germany, Dallas USA, Zurich Switzerland, Tokyo Japan
- url: https://www.whmcs.com/pricing/
- source_type: primary
- published: 2026
- confidence: high

### [9] WHMCS includes built-in provisioning modules for hosting control panels (cPanel/WHM, Plesk, etc.), domain registration/synchronization, SSL automation, and a Software Licensing addon; Blesta has native multi-company support (manage multiple brands from one installation) which WHMCS lacks natively.
- quote (WHMCS): "SSL Automation New" / "Software Licensing" addon
- url: https://www.whmcs.com/
- source_type: primary
- confidence: high
- quote (Blesta): "Native Multi-Company Support - Manage unlimited companies from a single installation with isolated settings, branding, and databases"
- url: https://www.blesta.com/compare/blesta-vs-whmcs/
- source_type: primary
- confidence: high

### [10] WHMCS has a larger third-party ecosystem (marketplace.whmcs.com) with more add-ons and themes, but many are paid; Blesta ships 26+ built-in plugins/modules, reducing dependency on third-party purchases.
- quote (Blesta): "Total Built-in Plugins: 26+" / "Many third party extensions can be found on The Marketplace. We also offer flat-rate sponsored development for most basic new modules and gateways, $750, and $600 respectively."
- url: https://www.blesta.com/integrations/
- source_type: primary
- confidence: high

### [11] Both WHMCS and Blesta have a hook/event system for provisioning automation — WHMCS uses action hooks and module hooks, while Blesta uses its event-based hook system with inbound+outbound webhooks, making Blesta more suitable for external webhook-driven provisioning flows.
- quote (WHMCS): Hook system documented at developers.whmcs.com/hooks/ with action hooks and module hooks.
- url: https://developers.whmcs.com/hooks/
- source_type: primary
- confidence: high
- quote (Blesta): "Inbound + Outbound" webhooks listed as a feature differentiator.
- url: https://www.blesta.com/compare/blesta-vs-whmcs/
- source_type: primary
- confidence: medium (vendor-comparison source)
